## Setup 

### **1. Clone this repository**

```bash
git clone https://github.com/Soham204/Expense-Tracker.git
cd expense-tracker-salesforce
```

### **2. Authorize your org**

```bash
sf org login web --alias ExpenseTrackerOrg
```

### **3. Push source**

```bash
sf project deploy start
```
